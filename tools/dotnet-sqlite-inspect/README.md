# dotnet-sqlite-inspect

Companion CLI tool for [SqliteInspector.Maui](../../src/SqliteInspector.Maui/) — handles platform detection, port forwarding, and browser launch for mobile SQLite database inspection.

## Usage

```bash
dotnet tool install -g dotnet-sqlite-inspect
sqlite-inspect
```

See the [SqliteInspector.Maui README](../../README.md) for full documentation.
